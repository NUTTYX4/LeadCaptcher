# LeadCaptcher

## Project Structure
- `frontend/`: Subhra's Workspace - Chat widget, prompts, mock data.
- `backend/`: Your Workspace - n8n workflows, db schemas, external APIs, docker-compose.
- `shared-contracts/`: Neutral Zone - Shared JSON schemas.
- `docs/`: Project documentation.

## Setup
1. Clone the repository
2. Run `docker-compose up -d` in the backend directory to start PostgreSQL and n8n.
